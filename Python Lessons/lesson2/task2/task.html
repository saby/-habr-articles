Задача 2

Звездный треугольник. Вывести N строк из «*», причем каждая строка должна быть пронумерована и длина строки равна номеру.

Входные данные:

  • N-количество строк(int)

Выходные данные:

  • Набор строк с номером строки и символами «*» (list)

Пример:

task2(3):
string_1 = '1 *'
string_2 = '2 **'
string_3 = '3 ***'
return [string_1, string_2, string_3]

Подсказки:

Чтобы получить нужную длину из "*", пользуйтесь возможностью "умножения" строк.
