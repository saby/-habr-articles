Задача 1

Используя модуль datetime, написать функцию change_month, которая к переданной дате прибавляет/вычитает переданное кол-во месяцев.

Входные данные:

  • Два аргумента. Дата (str) в формате "dd.mm.yy". Количество месяцев (int)

Выходные данные:

  • Дата (str) в формате "dd.mm.yy"

Пример:

Для сложения:
change_month('12.12.12', 7):
return '12.07.13'
Для вычитания:
change_month('01.11.10', -5):
return '01.06.10'

Подсказки:

Поминте, что в месяцах различное количество дней
